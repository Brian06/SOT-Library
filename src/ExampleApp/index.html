Browser Functions

isMobileDevice()
  • Does not need parameters
  • Returns boolean True value if we are in a mobile device
isTabletDevice()
  • Does not need parameters
  • Returns boolean True value if we are in a tablet device
isMobileOrTablet()
  • Does not need parameters
  • Returns boolean True value if we are in a mobile or in a tablet device
setReloadPageCachePersist()
  • Does not need parameters
  • set a pageshow event that will restart the page if there is cache saved

Validations Functions

validCVV("658")
  • Param 1 must be a string
  • Returns true if the string has 3 or 4 characters
atLeastOneLetter("John")
  • Param 1 must be a string
  • Returns true if the string has at least one letter
noEmptySpacesOnly("John")
  • Param 1 must be a string
  • Returns false if the string has just spaces
emptyOrletters("John")
  • Param 1 must be a string
  • Returns true if the string is empty or if just have letters

Miscellaneous Functions

googleAnalyticsTracking("Subscribe Page")
  • Param 1 must be a string, is the event label
  • Param 2 is optional, is an object with props
  • Register an event in google analytics
googleAnalyticsTrackingCustom("Button Click", "vin lookup", "Loading screen")
  • Param 1 must be a string, is the event label
  • Param 2 is the event category
  • Param 3 is the event action
  • Register an event in google analytics
